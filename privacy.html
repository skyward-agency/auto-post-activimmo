Politique de confidentialité

Dernière mise à jour : 23 décembre 2024

Éditeur : SWA INTERNATIONAL LTD

Service : ActivImmo Social Connect

Contact : contact@skyward-agency.ai

1. Données collectées

Notre application collecte uniquement les tokens d'accès nécessaires pour publier du contenu sur vos réseaux sociaux professionnels :

  • Facebook & Instagram : Tokens d'accès aux pages et comptes Business
  • LinkedIn : Tokens d'accès au profil et pages professionnelles
  • Twitter/X : Tokens d'accès au compte

2. Utilisation des données

Les tokens d'accès sont utilisés exclusivement pour publier du contenu sur vos comptes et pages selon vos instructions via notre plateforme ActivImmo. Aucune autre utilisation n'est effectuée.

3. Stockage des données

Les tokens sont stockés de manière sécurisée dans une base de données chiffrée (Supabase). Nous ne partageons jamais vos données avec des tiers. Les données sont conservées tant que vous utilisez notre service.

4. Sécurité

Nous utilisons des protocoles de sécurité standard de l'industrie pour protéger vos données. Les connexions sont chiffrées via HTTPS et les tokens sont stockés de manière sécurisée.

5. Vos droits

Vous pouvez à tout moment :

  • Révoquer l'accès depuis les paramètres de chaque plateforme (Facebook, LinkedIn, Twitter/X)
  • Demander la suppression de vos données en nous contactant
  • Accéder à vos données stockées

6. Réseaux sociaux concernés

Notre service se connecte aux plateformes suivantes :

  • Facebook & Instagram - via Facebook Graph API
  • LinkedIn - via LinkedIn API
  • Twitter/X - via Twitter API

7. Modifications

Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page avec une date de mise à jour.

Pour toute question concernant cette politique de confidentialité ou vos données :
